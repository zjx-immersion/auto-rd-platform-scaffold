## 🤝 贡献指南

### 如何更新文档

1. **主项目文档是源头**
   - 所有文档更新应先在主项目 `/docs/` 中进行
   - 然后同步到 scaffold

2. **同步到scaffold**
   ```bash
   # 在主项目根目录执行
   cp docs/规划/*.md auto-rd-platform-scaffold/docs/规划/
   cp docs/设计规范/*.md auto-rd-platform-scaffold/docs/设计规范/
   ```

3. **提交变更**
   ```bash
   git add auto-rd-platform-scaffold/docs/
   git commit -m "docs: 更新scaffold文档"
   ```
